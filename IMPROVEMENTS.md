# خطة تحسين وتطوير مشروع "سوشي أند جريل"

يحتوي هذا الملف على مقترحات تفصيلية لتحسين تطبيق الويب الخاص بمطعم "سوشي أند جريل" ونقل المشروع من مرحلة التصميم (Prototype) إلى مرحلة الإنتاج (Production).

## 1. تحسينات تجربة المستخدم واجهة الزبون (UX/UI)
- [ ] **تخصيص الأصناف (Product Customization):**
  - إضافة خيارات (Modifiers) لكل وجبة مثل (بدون بصل، إضافة صوص إضافي، نوع الأرز).
- [ ] **نظام تتبع الطلبات (Order Tracking):**
  - شاشة توضح للزبون المرحلة التي يمر بها طلبه (تم الاستلام -> قيد التحضير -> خرج للتوصيل).
- [ ] **ميزة "المفضلة" (Wishlist):**
  - السماح للزبون بحفظ أصنافه المفضلة لسرعة الطلب لاحقاً.
- [ ] **محرك بحث سريع:**
  - إضافة شريط بحث في المنيو مع فلاتر ذكية (مثلاً: نباتي، حار، الأكثر مبيعاً).

## 2. التحسينات التقنية والأساسية (Technical)
- [ ] **ربط قاعدة بيانات (Backend Integration):**
  - استبدال التخزين المؤقت (In-memory) بقاعدة بيانات حقيقية (مثل MongoDB أو PostgreSQL).
  - استخدام Supabase أو Firebase كحل سريع وسهل لإدارة البيانات والطلبات.
- [ ] **نظام المصادقة (Authentication):**
  - تسجيل دخول عبر الهاتف أو جوجل لحفظ بيانات العناوين وتاريخ الطلبات.
- [ ] **استمرارية البيانات (Persistence):**
  - تفعيل `persist` في Zustand لضمان عدم ضياع السلة عند تحديث الصفحة (Refresh).
- [ ] **تعدد اللغات (i18n):**
  - دعم كامل للتبديل بين العربية والإنجليزية لجميع نصوص الواجهة.

## 3. تحسينات لوحة الإدارة (Admin Panel)
- [ ] **التنبيهات الفورية (Live Notifications):**
  - استخدام WebSockets لتنبيه المدير فور وصول طلب جديد بصوت تنبيه.
- [ ] **تقارير وإحصائيات (Analytics):**
  - رسم بياني يوضح المبيعات اليومية والشهرية وأكثر الأصناف طلباً.
- [ ] **إدارة المخزون (Inventory Management):**
  - إمكانية إيقاف صنف معين يدوياً أو تلقائياً عند نفاذ الكمية.
- [ ] **تعديل المنيو:**
  - واجهة لرفع صور المنتجات وتعديل الأسعار والوصف دون الحاجة لتعديل الكود.

## 4. الميزات التجارية والتسويقية (Business)
- [ ] **أكواد الخصم (Promo Codes):**
  - نظام لإدارة وتطبيق خصومات على إجمالي الفاتورة.
- [ ] **برنامج الولاء (Loyalty Program):**
  - كسب نقاط مع كل طلب يمكن استبدالها بخصومات.
- [ ] **بوابات الدفع الإلكتروني:**
  - ربط خدمات دفع مثل Stripe أو Paymob لدفع الفواتير أونلاين.
